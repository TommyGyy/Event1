# 一号链接倒计时页面 - 设计文档

## 概述

移动端优先的单页面应用，提供可自定义的倒计时和活动列表展示。纯静态 HTML 文件，零依赖，手机浏览器直接打开即用。

## 页面结构

```
┌─────────────────────────────┐
│ 标题：一号链接倒计时          │
├─────────────────────────────┤
│                             │
│      02 : 30 : 00           │  ← 大号倒计时数字
│      时    分    秒          │
│                             │
│           ⚙ 设置            │  ← 点击弹出设置面板
│      🌙 暗夜舞台  [开关]    │  ← 主题切换
│                             │
├─────────────────────────────┤
│  🎆 烟花秀          ✏ ✕   │
│  🎵 电音节          ✏ ✕   │  ← 活动列表（可编辑/删除）
│  ⭐ 星空大合唱      ✏ ✕   │
│  💡 灯光花海        ✏ ✕   │
│  🔦 山体激光秀      ✏ ✕   │
├─────────────────────────────┤
│        + 添加活动           │  ← 新增按钮
└─────────────────────────────┘
```

## 设置面板（弹出浮层）

- 时/分/秒三个输入框，数字键盘输入
- 主题切换：暗夜舞台 / 篝火暖色，即时生效
- 三个控制按钮：开始 / 暂停 / 重置
- 点击面板外部或关闭按钮收起

## 活动列表管理

- 点击活动文字 → 进入编辑模式（内联编辑或弹窗）
- 点击 ✕ → 删除该活动
- 底部 "+ 添加活动" → 新增空白项并进入编辑

## 视觉风格

### 暗夜舞台风（默认）
- 背景：深黑紫色渐变 `#0a0a1a → #1a0a2e`
- 数字颜色：霓虹紫 `#c084fc`，带发光效果 text-shadow
- 卡片底色：半透明深紫
- 强调色：紫/品红渐变

### 篝火暖色风
- 背景：深棕暖色渐变 `#1a0a00 → #331100`
- 数字颜色：琥珀橙 `#f59e0b`，带火光发光效果
- 卡片底色：半透明暖棕
- 强调色：橙/金渐变

主题切换通过 CSS 类名切换实现，所有颜色使用 CSS 自定义属性。

## 数据持久化

使用 localStorage 存储：
- 倒计时时长（h, m, s）
- 当前主题选择
- 活动列表内容
- 倒计时运行状态（运行中/暂停、剩余秒数）

页面加载时从 localStorage 恢复状态。

## 技术实现

- 单 HTML 文件（~300 行）
- 内嵌 CSS（CSS 自定义属性驱动主题）
- 内嵌 JS（原生 DOM 操作，无框架）
- 倒计时基于 setInterval，每秒更新
- 移动端适配：viewport meta，触屏友好的按钮尺寸（≥44px）

## 不在范围内

- 音效
- 倒计时结束后的自动动作
- 后台持续运行（切换 App 后计时可能偏移）
- 多设备同步
